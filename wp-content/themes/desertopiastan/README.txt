DAILY DISH PRO THEME
https://my.studiopress.com/themes/daily-dish/

INSTALL
1. Upload the Daily Dish Pro theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Daily Dish Pro theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking. To set up the theme like the demo, please visit https://my.studiopress.com/documentation/daily-dish-theme/.

WIDGET AREAS
Before Header - Widgets in this section will display before the header on every page.
Home Top - Widgets in this section will display at the top of the homepage.
Home Middle - Widgets in this section will display in the middle of the homepage.
Home Bottom - Widgets in this section will display at the bottom of the homepage.
Primary Sidebar - This is the primary sidebar if you are using the Content/Sidebar, Sidebar/Content, Content/Sidebar/Sidebar, Sidebar/Sidebar/Content or Sidebar/Content/Sidebar Site Layout option.
Secondary Sidebar - This is the secondary sidebar if you are using the Content/Sidebar/Sidebar, Sidebar/Sidebar/Content or Sidebar/Content/Sidebar Site Layout option.
After Entry - This is the widget that appears after the entry on single posts.
Before Footer Widgets - Widgets in this section will display before the footer widgets on every page.
Footer #1 - Widgets in this section will display in the first column of the footer section of every page.
Footer #2 - Widgets in this section will display in the second column of the footer section of every page.
Footer #3 - Widgets in this section will display in the third column of the footer section of every page.
After Footer - Widgets in this section will display at the bottom of every page.

CUSTOM BACKGROUND
To change the Daily Dish Pro background image, go to Appearance > Customize > Background Image to upload your desired image file.
The texture that was used in the Daily Dish Pro theme are courtesy http://subtlepatterns.com/.
